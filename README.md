## Дисклеймер

Инструмент работает только с открытыми данными и чатами, где состоит ваш аккаунт.  
Используйте его в рамках законодательства вашей страны и правил Telegram.

# Telegram OSINT TXT Tool

Простой инструмент для OSINT‑аналитики Telegram‑аккаунтов и каналов.  
Собирает базовую информацию о профиле и публичные сообщения цели, а результаты сохраняет в текстовые файлы в папке `output/`.

## Возможности

- Снимок профиля пользователя или канала (`snapshot-profile`):
  - ID
  - username
  - имя / название
  - тип (user / channel)
  - время снимка
- История профиля (`show-profile-history`):
  - выводит все ранее сделанные снимки цели
- Сбор публичных сообщений (`collect-public`):
  - ищет сообщения цели во всех каналах/группах, где состоит ваш аккаунт
  - сохраняет сообщения в txt‑файл
- Поиск по сообщениям (`search-public`):
  - фильтр по ключевым словам по ранее собранным сообщениям

Все данные хранятся в `.txt`‑файлах, базы данных не используются.

## Требования

- Python 3.10+
- Аккаунт Telegram с:
  - `api_id` и `api_hash` с https://my.telegram.org  
  - доступом к коду подтверждения и, при необходимости, паролю двухэтапной аутентификации
- Библиотеки Python:
  - `telethon`
